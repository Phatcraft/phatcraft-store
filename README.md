# Phatcraft Store
Phatcraft Store là một dự án cá nhân được phát triển và quản lý bởi Phatcraft.<br>
Dự án được định hướng như một khu vực phát triển các hệ thống của Phatcraft.
> [!CAUTION]
> _Hiện tại dự án vẫn hoạt động local và không có dự tính cho việc phát triển ra bên ngoài_

## Thông tin dự án
### 1. Thông tin cơ bản
| Tên thông tin  | Nội dung                        |
|---------------:|:-------------------------------:|
| Tên dự án      | Phatcraft Store                 |
| Creator        | Phatcraft                       |
| Quản lý dự án  | Phatcraft                       |
| Ngày tiến hành | 14/6/2026                       |
| Định hướng     | Website bán hàng online cá nhân |

### 2. Các thành phần dự án
| Tên thành phần   | Tên dịch vụ          | Ngôn ngữ |
|-----------------:|:--------------------:|:--------:|
| Thiết kế website | Express JS           | NodeJS   |
| Web server       | NGINX                |          |
| Database         | MariaDB              | SQL      |
| Quản lý dịch vụ  | Docker               |          |
| Phần cứng server | Raspberry Pi Zero 2W |          |
