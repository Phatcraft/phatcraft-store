# Phatcraft Store

## 1. Về dự án
Phatcraft Store là một dự án website bán hàng online cá nhân của Phatcraft.<br>
Dự án được định hướng như một không gian thử nghiệm công nghệ cá nhân của Phatcraft.

## 2. Thông tin dự án
+ **Tên dự án:** Phatcraft Store
+ **Creator & quản lý:** Phatcraft
+ **Ngày thành lập:** 1/7/2026
+ **Công nghệ sử dụng:**

  | Tên hạng mục | Tên công nghệ sử dụng | Ngôn ngữ |
  |:------------:|:---------------------:|:--------:|
  | Thiết kế website | Slim Framework | PHP |
  | Quản lý web server | NGINX |
  | Quản lý database | MariaDB| SQL |
  | Quản lý ứng dụng | Docker |
  | Phần cứng server | Raspberry Pi Zero 2W |
